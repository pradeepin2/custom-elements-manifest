Simple example of Custom element manifest representation of an web component

How to use
To run this server

clone or download the project
cd into the custom-elements-manifest folder and use
npm install
npm start
The server will start at http://127.0.0.1:3000

Note: 
1. Custom element custom-button will have custom elements manifest which JSON representation UI components.
2. The JSON can be included in an in a documentation viewer of a web component like API Viewer is a set of custom elements and helpers providing interactive UI for documenting web components.
